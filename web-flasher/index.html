After Flashing — First-time Setup

  1. Open a serial monitor at 115200 baud (e.g. idf.py monitor or PlatformIO).
  2. Add WiFi: !wifi 732-50-FUBAR aquaman13
  3. Set server (use the laptop's IP on the event WiFi): !server 192.168.X.X
  4. Device reboots and connects automatically — you'll see "Connecting…" then "Ready".
  5. Speak to it — or type a message in the serial monitor and press Enter.

Type !help in the serial monitor for all available commands. Type !status to see the current WiFi and server.
